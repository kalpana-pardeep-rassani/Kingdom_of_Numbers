# Kingdom of Numbers - Demo Video Script

## Demo Duration

5 to 7 minutes

## 1. Introduction

Assalam-o-Alaikum. My name is [Your Name], and this is my final graduation project, Kingdom of Numbers.

Kingdom of Numbers is a gamified math learning web app designed for Grade 6 to 8 students studying under the Federal Board Pakistan curriculum. The goal of this project is to make mathematics more engaging, less stressful, and more personalized by turning practice into a story-based adventure.

## 2. Problem Statement

Many students struggle with mathematics because traditional learning methods are repetitive and not interactive. Students often experience math anxiety, low motivation, and do not receive immediate feedback or a personalized path based on their weak areas.

This project solves that problem by combining math practice with game mechanics such as worlds, levels, characters, rewards, and progress tracking.

## 3. Solution Overview

In this app, the student becomes a hero inside the Kingdom of Numbers.

The app includes:

- Signup and login system
- Diagnostic test
- Result analysis with weak and strong topics
- Personalized world recommendations
- Four story-based math worlds
- Instant feedback and explanations
- XP points, achievements, and streaks
- Progress dashboard and profile management
- Local Storage data persistence

## 4. Show Login and Signup

Now I will start the demo.

First, the student can create a new account by entering their name, class, email, password, and favorite world.

The app validates the form fields, such as required values, email format, and password length.

After signup, the account is saved in browser Local Storage and the student is taken directly to the diagnostic test.

## 5. Show Diagnostic Test

This is the diagnostic quest screen.

It includes 10 questions from integers, fractions, algebra, and geometry with easy, medium, and hard difficulty.

The purpose of this test is to analyze the student’s current level and identify weak topics.

After submitting the answers, the app calculates the score topic by topic.

## 6. Show Diagnostic Result Analysis

This is the result analysis page.

Here the app shows:

- overall score
- weak topics
- strong topics
- topic percentages
- recommended world order

This creates a personalized learning path, which is one of the main goals of the project.

## 7. Show Worlds and Gameplay

Now I will start the recommended practice.

The app contains four main worlds:

- Integers Island
- Fractions Forest
- Algebra Academy
- Geometry Garden

Each world contains 10 levels, and each level contains one question.

Every question is wrapped in a story scenario so it feels like a game instead of a normal quiz.

The student also sees character-based dialogue from the Math Wizard and the villain of that world.

When the student answers correctly, the app gives:

- XP points
- possible speed bonus
- instant explanation
- achievement unlocks if conditions are met

If the answer is incorrect, the app shows feedback and allows the student to try again.

## 8. Show Dashboard and Progress

This is the dashboard.

Here the student can see:

- total XP
- current streak
- accuracy
- achievement count
- world progress
- personalized recommendations

This progress view makes learning visible and motivating.

Now I will open the progress page.

This screen shows topic mastery, world completion, and earned badges.

## 9. Show Profile and CRUD

Now I will open the profile page.

Here the student can update their profile information such as name, grade, and favorite world.

This demonstrates update functionality.

The app also supports reset progress and delete local account, which demonstrate delete operations.

So the project includes full CRUD using Local Storage:

- Create user account
- Read saved progress and profile
- Update profile and stats
- Delete or reset saved data

## 10. Technical Implementation

This project was built using:

- HTML5
- CSS3
- Vanilla JavaScript
- Browser Local Storage

I selected this stack because it is simple, cost-effective, easy to understand, and suitable for a student-level MVP without requiring a backend server.

## 11. Conclusion

In conclusion, Kingdom of Numbers is a practical and engaging solution for improving mathematics learning for middle school students.

It addresses the problems of boredom, low motivation, lack of personalization, and poor progress visibility by transforming math into a playful learning journey.
